Tamanho da Imagem de fundo

Há várias formas de redimensionar a imagem para força-lá a caber na caixa. Por padrão, nenhum redimensionamento será aplicado, e a imagem será exibida do seu tamanho natural. Porém, podemos usar propriedades background-size para alterar esse comportamento.

  • background-size: auto - (padrão) a imagem do fundo será aplicada em seu tamanho original
  • background-size: px ou % - Redimensiona a largura da imagem e faz altura se adaptar automaticamente. Podemos também informar as duas dimensões na sequência ou também usar valores percentuais.
  • background-size: cover - Muda o tamanho da imagem para que ela seja sempre totalmente exibida na tela, sem nenhum corte.
  • background-size: contain - Redimensiona a imagem para que ela cubra o contêiner, mesmo que para isso ocorram alguns eventuais cortes.

Dentre todas as opções a melhor que se destaca é a cover já que ela vai tentar ser fiel ao tamanho original da imagem sem distorcer.

Outra propriedade que possuimos alem da Size seria a attachment (vínculo). Ela tem como função de vincular a imagem de fundo com o resto do documento, principalemente se o conteúdo for maior do que a altura da página e seja necessário vazar uma rolagem vertical

Algumas propriedades do attachment são:

  • background-attachment: Scroll - (padrão) a imagem do fundp vai rolar junto com o conteúdo
  • background-attachment: Fixed - A imagem de fundo vai ficar fixada enquanto o conteúdo vai ser rolado

*A melhor combinação para backgound seria a background-size: contain com background-attachment: Fixed

Para facilitar todos esse códigos, podemos utilizar uma Shorthand, segundo essa sequência de códigos:

"background: [color] [image] [position] [repeat] [size] [attachment];"

*A forma do size pode não funcionar na maioria dos navegadores, ou seja, é mais fácil criar a propria tag para o size do que incluir na Shorthand
